# 卡面美化 / CardBeautify

《Slay the Spire 2 / 杀戮尖塔2》卡牌立绘替换 MOD，提供适合手机触控的百科大全卡面切换按钮，并记住每张卡上次选择的卡面。

## 最新版本

- [v0.4.9](https://github.com/Yummn/sts2-card-beautify/releases/tag/v0.4.9)：Android 启动稳定性修复。删除旧版 6 个 `NCard` Harmony 钩子，仅保留 1 个卡图属性钩子；百科按钮由场景监听器维护。

## 安装

下载 Release 中的 `CardBeautify-v0.4.9.zip`，解压后将 `CardBeautify` 文件夹放入游戏 `mods/` 目录。压缩包已包含 `CardBeautifyAssets.pck`。

## v0.4.9 验证

- REDMI K80 Pro、Android v0.103.2 实机验证。
- 与 BetterDefect v0.8.6、TifiraDefectSkin v1.1.7 等整套模组连续冷启动 5/5 成功。
- 百科大全可见卡位 25 个，其中 18 张可切换卡面，18 个按钮均正常生成。
- 卡面切换、BetterDefect 禁用/升级按钮和动态出率文字可同时显示。
