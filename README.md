# 卡面美化 / CardBeautify

Slay the Spire 2 card portrait beautify mod with mobile-friendly controls.

这些是给《Slay the Spire 2 / 杀戮尖塔2》v103/v107 调试制作的 MOD 归档。可安装压缩包放在 GitHub Releases；如果有多个版本，Release 按旧版到新版保留。

## 下载

- [v0.4.1](https://github.com/Yummn/sts2-card-beautify/releases/tag/v0.4.1)
- [v0.4.2-PC-v107](https://github.com/Yummn/sts2-card-beautify/releases/tag/v0.4.2-PC-v107)
- [v0.4.2-Mobile-v103](https://github.com/Yummn/sts2-card-beautify/releases/tag/v0.4.2-Mobile-v103)
- [v0.4.6](https://github.com/Yummn/sts2-card-beautify/releases/tag/v0.4.6)
- [v0.4.7](https://github.com/Yummn/sts2-card-beautify/releases/tag/v0.4.7)（最新版：修复 BetterDefect 还原的一代机器人卡面加载）

## 安装

下载对应 Release 里的 zip，解压后把其中的 CardBeautify 文件夹放入游戏 mods 目录。手机版 v103 可直接使用 v0.4.7。

## v0.4.7 说明

- 卡面美化本体继续使用 CardBeautifyAssets.pck。
- 对 BetterDefect 新增/还原的一代机器人卡，如果美化包里没有对应条目，会从同级 BetterDefect/Data/Portraits/*.png 读取正确的一代卡图。
- 启动日志会显示 BetterDefect portrait folder ... png=26，用于确认手机端资源已安装。

## 备注

- 最新本地整合包来自 C:\Users\yummn\Downloads\杀戮尖塔2MOD\可安装-已测试v103。
- 旧版本仅作留档；通常建议使用最新版本。
