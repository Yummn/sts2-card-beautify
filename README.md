# 卡面美化 / CardBeautify

Slay the Spire 2 card portrait beautify mod with mobile-friendly controls.

这些是给《Slay the Spire 2 / 杀戮尖塔2》v103/v107 调试制作的 MOD 归档。可安装压缩包放在 GitHub Releases；如果有多个版本，Release 按旧版到新版保留。

## 安装

下载对应 Release 里的 zip，解压后把其中的 $(@{Id=CardBeautify; Repo=sts2-card-beautify; Name=卡面美化 / CardBeautify; Desc=Slay the Spire 2 card portrait beautify mod with mobile-friendly controls.; Versions=System.Object[]}.Id) 文件夹放入游戏 mods 目录。
